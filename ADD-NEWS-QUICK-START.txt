METHOD 2: Admin Panel (For Custom Content)
──────────────────────────────────────────
1. Start servers: npm start (Backend) & npm run dev (Frontend)
2. Go to: http://localhost:5173/admin/login
3. Click "Articles" → "Create New"
4. Fill form and save
5. Repeat for each article

METHOD 3: Bulk Import from JSON
────────────────────────────────
1. Create articles.json with your data
2. Create import script
3. Run: node import-articles.js

🔥 BREAKING NEWS vs FEATURED:
═══════════════════════════════════════════════════════════════════════

Breaking News (isBreaking: true)
   → Shows in top carousel
   → Red badge
   → Auto-rotates

Featured (isFeatured: true)
   → Shows in featured grid
   → Star badge
   → 3-column layout

Regular Articles
   → Shows in "Latest News"
   → All published articles
